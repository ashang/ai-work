# Game Controllers

This directory contains the game controller and orchestration logic.

## Key Components

- **GameController**: Main controller orchestrating game flow
- **Turn Management**: Alternating between player and AI turns
- **State Coordination**: Managing state between engine, AI, and UI
- **Game Lifecycle**: Starting, ending, and resetting games

## Responsibilities

- Coordinate between chess engine and AI opponent
- Handle player move requests
- Trigger AI move generation
- Manage game state transitions
- Provide game state to UI components
