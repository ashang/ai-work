# Chess Engine

This directory contains the chess engine implementation responsible for:

- Move validation and generation
- Game state management
- Check, checkmate, and stalemate detection
- Special moves (castling, en passant, pawn promotion)
- Board representation and manipulation

## Key Components

- **ChessEngine**: Main engine class implementing move validation and game rules
- **Move Generation**: Logic for generating valid moves for each piece type
- **Game State**: Board state tracking and turn management
- **Rule Enforcement**: Implementation of all standard chess rules
