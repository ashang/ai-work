# User Interface

This directory contains all UI components for the chess application.

## Key Components

- **Board Rendering**: 8x8 chess board with pieces
- **Piece Selection**: Interactive piece selection and move highlighting
- **Move Animation**: Smooth piece movement animations
- **Game Information**: Move history, captured pieces, turn indicator
- **Game Controls**: New game, resign, difficulty selection
- **Theme System**: Light/dark mode with Hugging Face-inspired design

## Features

- Responsive design (320px - 2560px)
- Touch and mouse event support
- Visual feedback for valid moves
- Game state display
- Theme switching with persistence
