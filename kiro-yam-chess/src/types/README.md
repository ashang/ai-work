# Type Definitions

This directory contains TypeScript type definitions and interfaces used throughout the application.

## Key Types

- **Position**: Board position (row, col)
- **Piece**: Chess piece with type, color, and position
- **Move**: Move representation with source, destination, and metadata
- **GameState**: Complete game state including board, turn, and status
- **Theme**: Theme configuration and color schemes
- **AI Config**: AI difficulty and configuration settings

## Purpose

Centralized type definitions ensure type safety and consistency across all components.
All types follow the interfaces defined in the design document.
